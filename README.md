# dotfiles

## Initial setup

```
$ git clone git://github.com/skyriser/dotfiles.git
$ cd dotfiles
$ make
```

## Manually install contents

### Softwares

- [Homebrew](https://brew.sh/)
- [Starship](https://starship.rs/)
- [exa](https://github.com/ogham/exa)
- [anyenv](https://github.com/anyenv/anyenv)

### nvim plugins

Run first time only

```
call dein#install()
```
